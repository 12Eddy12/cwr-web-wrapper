Music Metadata by Matija Kolarić is a software library written in pure Python.

It is a sponsored open-source project. Sponsors get access to features they sponsor immediately. As cost of development are covered, features are released as open-source (MIT license).

Open-source code repositories

Open-source code is available at github.com/musicmetadata.

At this moment, this includes:

  • Web wrapper - the site you are currently visiting,
  • Territories - features related to territory calculations, 1 demo,
  • EDI - underlying format for all CWR and CRD versions, 2 demos.

Completed features available to sponsors

These features have been developed and thoroughly tested, and can be made available to sponsors at once:

  • CWR 2.x validation and parsing - Common Works Registration, versions 2.1 and 2.2

5 demos are available on this site.

Features in various stages of development

These features are being actively developed and tested.

  • CWR 2.x generation (with validation) - Common Works Registration, versions 2.1 and 2.2 - alpha,
  • CWR 3.x parsing, validation and generation - Common Works Registration, versions 3.0 and 3.1 - half-way
  • EBR - Electronic Batch Registration, in alpha
  • CRD - Common Royalty Distribution, in early stages of development
  • DDEX - in early stages of development, MWN, RIN and ERN will be coming out first.
