This is the default home page for this site. It's purpose is to provide some web-based experience of several music metadata libraries written in Python.

Although the demo is targeting a much wider audience, the libraries are to be used by developers.

These libraries will provide open source replacements for the tools available at matijakolaric.com, depending on the sponsors.

Feel free to become a sponsor and then inquire about timelines.
