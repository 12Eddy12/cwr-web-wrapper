This is the web wrapper. It is available as open-source code at github.com/musicmetadata/web-wrapper.

It's purpose is to provide:

  • Simple installation - anyone can deploy web wrapper and all open-source elements of the software library,
  • Web interface - features provided in the demos can be tested by anyone, without help of developers and/or system engineers,
  • Demos - by providing examples of most common use cases, anyone can see if the library would work for them.

Open-source or everything?

Web wrapper also supports features not yet available as open source. Sponsors can install demos of all available features.
